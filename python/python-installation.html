Python Installation

Here you can see how to install Python 3 on Debian Linux, macOS or Windows.
Moreover we install the Python Development Environment "PyCharm Community Edition". It's free, but pretty helpful.
But you don't need PyCharm. You can also use IDLE or something else.

Debian

Open the terminal and type:

sudo apt-get install python3

If you have Ubuntu, it's suggested to install PyCharm Community Edition. Type:

sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make
sudo apt-get update
sudo apt-get install ubuntu-make

for getting the the latest version of PyCharm Community Edition.

macOS

Go to the official website of Python, download the latest version and install it.

Now you have Python 3 installed on your machine, download the latest version of PyCharm Community and install it.

Windows

Go to the official website of Python, download the latest version and install it.

Now you have Python 3 installed on your machine, download the latest version of PyCharm Community and install it.
